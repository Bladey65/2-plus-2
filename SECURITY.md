## Reporting a Vulnerability

The project maintainers take the structural integrity of execution architectures seriously. If you believe you have discovered a security vulnerability, code injection vector, or unexpected operational flaw within this repository, please adhere to the following protocol:

1. Do not disclose the vulnerability through a public GitHub issue or pull request.
2. Compile a detailed report containing a description of the flaw, reproduction steps, and an environment dump.
