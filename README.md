\# Why Did I Make This? (The 2 + 2 Project)



This repository is a monument to procrastination. It is an incredibly over-engineered, multi-file software suite designed to solve a math problem that a literal toddler can solve using their fingers. 



Yes, it requires \*\*Python 3.14.6\*\*. No, it does not do anything useful. 



\*(Full disclosure: Every single piece of code in this repository, including this existential crisis of a README, was generated by Google Gemini because I wanted an AI to help me violently over-complicate basic arithmetic.)\*



\## 🧠 The "Why" (An Anatomy of Regret)



I had a perfectly good afternoon, and instead of doing something productive, I decided to treat `2 + 2 = 4` like a NASA rocket launch. 



I didn't just write a one-line script. I split it into a whole folder structure because apparently, I don't trust Python's internal math capabilities and needed a dedicated "Audit Bureau" script to make sure four is actually four.



\## 📁 The Folder Layout of Shame



```text

2 + 2/

├── README.md                              <-- This crisis of conscience

└── src/

&#x20;   ├── pre\_flight\_inspection.py           <-- Checks if the computer can handle the number 4

&#x20;   ├── constants.py                       <-- Hardcodes the number 2 so it doesn't escape

&#x20;   ├── two\_plus\_two\_but\_make\_it\_a\_space\_launch.py <-- The main script (mostly just dramatic countdowns)

&#x20;   ├── validator.py                       <-- Triple-checks that the AI didn't lie to me

&#x20;   ├── exceptions.py                      <-- In case the universe collapses and 2+2=5

&#x20;   ├── historical\_database.py             <-- Writes the number 4 to a text file like a historical event

&#x20;   └── manual\_override\_abort.py           <-- In case I panic during basic addition

```



\## 🏃‍♂️ How to Run This Waste of CPU Cycles



If you actually want to waste 3 minutes of your life running this step-by-step from the root folder, here is how you do it:



\### Step 1: Over-analyze your computer

Make sure your multi-core modern CPU is mentally prepared to hold a single-digit integer:

```bash

python3 src/pre\_flight\_inspection.py

```



\### Step 2: The actual space launch

Run the main script. It will literally count down from 3 just to add two numbers together. Prepare for heavy logging overhead for a basic primary school problem:

```bash

python3 src/two\_plus\_two\_but\_make\_it\_a\_space\_launch.py

```



\### Step 3: Write it down so we never forget

Permanently archive the fact that we successfully achieved the number 4 into a text file, because god forbid we lose this groundbreaking data:

```bash

python3 src/historical\_database.py

```



\## 🚨 The Panic Button

If the code starts sweating or you get scared that basic arithmetic is going to break reality, quickly run this to kill the program and save your computer's dignity:

```bash

python3 src/manual\_override\_abort.py

```
